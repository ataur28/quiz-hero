Question And Answer

1. What is the difference between .map() .filter() .foreach() and .find() in javascript?

Answer:

.forEach(): This function is another form of just a plain for loop that can be used to iterate through array items.

.map(): This function returns a new array with modified values.

.filter():This function is a search function that returns all the elements that fulfil the assigned condition.

.find(): This function returns only one match in an array.

2. What is difference between LocalStorage SessionStorage and Cookies?

Answer:

LocalStorage:

  • 5MB/10MB storage
  • it's not session based, need to be deleted via JS or manually
  • Client side resding only
  • Less older browsers support

SessionStorage:

  • 5MB storage
  • it's session based, working per window or tab
  • Client side resding only
  • Less older browsers support

Cookie:

  • 4KB storage
  • Expiry depends on the setting and working oer window or tab
  • server and Client side resding only
  • More older browsers support

3. What is event queue and how it works?

Answer: Event Loop monitor the Call Stack and the Callback Queue.If the call stack is empty, the event loop will take the first event from the queue and push it onto the call stack, which effectively executes it. This type of repetition is called a tick.
